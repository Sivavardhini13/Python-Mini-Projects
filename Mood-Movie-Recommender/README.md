# 🎬 Mood-Based Movie Recommender

A simple Python mini project that recommends movies based on the user's current mood.

## 📌 Features

* Recommends movies for different moods.
* Supports multiple mood categories:

  * Happy
  * Funny
  * Romantic
  * Sad
  * Angry
  * Excited
  * Bored
  * Scared
  * Stressed
  * Relaxed
  * Nostalgic
* Easy-to-use command-line interface.
* Beginner-friendly Python project.

## 🛠️ Technologies Used

* Python 3

## 🚀 How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Run the program:

```bash
python app.py
```

4. Enter your mood when prompted.
5. The program will recommend a movie based on your input.
